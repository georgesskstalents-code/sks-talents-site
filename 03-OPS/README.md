## Purpose

This directory is the operational reference for the SKS TALENTS production site.
It explains which services the site depends on, which environment variables are required, and who to contact or where to look when a service needs rotation or verification.

## Files

- `notion.md` — Notion CMS and content sync
- `resend.md` — email routing via Resend
- `turnstile.md` — anti-bot configuration
- `calendly.md` — booking and scheduling setup
- `vercel.md` — hosting and deployment details
- `ovh.md` — DNS configuration for skstalents.fr
- `api-status.md` — project API and key status tracker
