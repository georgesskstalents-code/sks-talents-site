# Panorama — Documents de référence SKS Talents

Ce dossier contient les **panoramas** : documents de référence vérifiés (marché, personas, fonctions, écosystème) que Claude consulte quand le CEO dit "panorama".

## Convention

- **Format :** Markdown ou PDF.
- **Nommage :** `YYYY-MM-DD-titre-court.md` (ex: `2026-05-06-personas-cibles.md`)
- **Mise à jour :** chaque fois qu'une donnée change, créer une nouvelle version datée. Les anciennes restent en archive.

## Catégories attendues

| Préfixe | Contenu |
|---|---|
| `personas-` | Cibles persona du site (CEO biotech Series A, COO scale-up, DRH groupement vétérinaire, etc.) |
| `fonctions-` | Fonctions couvertes par SKS (Account Manager Diagnostic, CFO biotech, etc.) |
| `marche-` | Données marché (taille TAM, concurrents, segmentation) |
| `ecosysteme-` | Carte écosystème (fonds, clusters, médias, écoles) |
| `process-` | Process internes (méthode 4 étapes, scoring candidat, etc.) |

## Comment Claude utilise ce dossier

Quand le CEO dit "panorama" ou "regarde le panorama" → Claude lit ce dossier en priorité avant de produire toute analyse stratégique.

Si un fichier est manquant ou daté de plus de 6 mois, Claude doit signaler l'obsolescence avant de s'appuyer dessus.
